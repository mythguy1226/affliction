# Affliction
A First-Person Survival Shooter made in Unreal Engine 5 utilizing both C++ and Blueprints. The project features a map for the player to fight endless waves of the undead in and progressively gets more difficult to survive. The player can gain points by killing enemies and can use those points to unlock different weapons, ammo refills, and new sections of the map.
